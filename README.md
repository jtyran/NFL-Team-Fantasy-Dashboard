# NFL-Team-Fantasy-Dashboard

A Power BI dashboard designed to provide a team-level fantasy football analytics experience. This project transforms raw football data into actionable insights for fantasy managers, analysts, and sports decision-makers by combining team tendencies, player opportunity metrics, and forward-looking fantasy tools.

Project Overview

The goal of this project was to build an interactive fantasy football dashboard that helps users evaluate NFL teams from a fantasy perspective rather than only a real-life football lens.

This dashboard focuses on questions such as:

Which teams create the most fantasy value?
Where are vacated targets available?
How are touches distributed across positions?
Which rushing gaps generate the most yards?
How does a team’s depth chart impact opportunity?
Which players or teams may be undervalued heading into 2026?

Objective:

Create a modern, interactive dashboard that allows fantasy football users to make smarter draft, roster, and weekly lineup decisions using team-based data analytics.
